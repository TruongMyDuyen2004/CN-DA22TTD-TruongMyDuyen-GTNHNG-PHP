## Lý do thay đổi

### 1. Xu hướng 2025
- Indigo/Purple là màu trending trong design hiện đại
- Được sử dụng bởi các brand lớn: Stripe, Twitch, Discord
- Tạo cảm giác tech-forward và innovative

### 2. Tâm lý màu sắc
- **Indigo**: Tin cậy, chuyên nghiệp, sang trọng
- **Purple**: Sáng tạo, cao cấp, độc đáo
- **Pink**: Năng động, thân thiện, hiện đại

### 3. Accessibility
- Contrast ratio tốt hơn
- Dễ đọc trên mọi background
- Phù hợp với WCAG 2.1 standards

### 4. Versatility
- Dễ kết hợp với nhiều màu khác
- Phù hợp cả light và dark mode
- Linh hoạt cho nhiều use cases

## Best Practices

### Do's ✅
- Sử dụng gradients cho elements quan trọng
- Kết hợp primary với secondary để tạo contrast
- Dùng neutral colors cho text và backgrounds
- Áp dụng status colors đúng ngữ cảnh
- Maintain consistency across all pages

### Don'ts ❌
- Không mix quá nhiều màu trong một element
- Không dùng pure black (#000000)
- Không bỏ qua accessibility
- Không overuse gradients
- Không dùng màu quá chói

## Accessibility

### Contrast Ratios
- **Text on White**: 
  - Gray 900: 16.1:1 ✅
  - Gray 700: 8.6:1 ✅
  - Gray 500: 4.6:1 ✅

- **White on Primary**:
  - Indigo 500: 8.6:1 ✅
  - Purple 500: 7.2:1 ✅
  - Pink 500: 5.8:1 ✅

### WCAG Compliance
- **AA**: Minimum 4.5:1 for normal text
- **AAA**: Minimum 7:1 for normal text
- All combinations meet AA standards
- Most combinations meet AAA standards

## Implementation

### CSS Variables
```css
:root {
    --primary: #6366f1;
    --primary-dark: #4f46e5;
    --secondary: #ec4899;
    --accent: #f59e0b;
    --success: #10b981;
    --danger: #ef4444;
    --warning: #f59e0b;
    --info: #3b82f6;
}
```

### Usage
```css
/* Button */
.btn-primary {
    background: var(--gradient-primary);
    color: white;
}

/* Badge */
.badge-success {
    background: linear-gradient(135deg, 
        rgba(16,185,129,0.1), 
        rgba(20,184,166,0.1));
    color: var(--success);
}

/* Text */
.text-primary {
    color: var(--primary);
}
```

## Migration Guide

### Updating from v1.0 to v2.0

1. **Replace color variables**
```css
/* Old */
--primary-color: #c62828;
--secondary-color: #ff6f00;

/* New */
--primary-color: #6366f1;
--secondary-color: #ec4899;
```

2. **Update gradients**
```css
/* Old */
linear-gradient(135deg, #c62828, #ff6f00)

/* New */
linear-gradient(135deg, #6366f1, #8b5cf6)
```

3. **Test all components**
- Buttons
- Badges
- Alerts
- Cards
- Forms

4. **Check accessibility**
- Run contrast checks
- Test with screen readers
- Verify color blindness compatibility
